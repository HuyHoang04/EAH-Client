Core Purpose
Provide an environment that allows instructors to automate processes without requiring in-depth IT knowledge. This report clarifies key architectural decisions.
Main objective
Set up a “playground” for instructors to automate routine administrative tasks and more complex pedagogical processes.
Core Platform
node-based system , allowing drag-and-drop and connection of action/integration “nodes” to build automation flows.
Automation scope (focus on instructors)
Class attendance : distribute attendance sheets (QR/link), summarize, and warn of absences.
Scoring & feedback : self-scoring multiple choice tests, suggested rubrics for essays, quick feedback.
Track learning progress : attendance/results dashboard, conditional reminders.
Notification : conditional mass text/email template.
Create & export documents (DOCX/PPTX/PDF) :
Create lesson plans, lecture slides, study sheets, rubrics, tests, and answers.
Export .docx / .pptx / .pdf , sync Google Docs/Slides and Microsoft Office .
Support templates (logo, brand colors), mail merge (class roll), automatic page numbering/table of contents.
Insert tables/charts from CSV/Google Sheets , embed images/videos.
Quickly generate slides from outlines/learning objectives : create slide frames, distribute content, suggest illustrations.
Question bank & shuffle test : generate multiple test codes, shuffle questions/answers, watermark, export to Word/PDF.
Collect and organize submissions : collect from Classroom/Moodle/Drive/Email, rename files, divide folders, create grading lists.
Teaching schedule & reminders : create personal timetable, remind deadlines/grades, synchronize Google Calendar/Outlook.
